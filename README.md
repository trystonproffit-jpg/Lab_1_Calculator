# Lab_1_Calculator A JavaScript Calculator That saves History

This is a basic calculator written in JavaScript. 
The calculator will allow you to add, subtract, multiply, and divide any two numbers.
The calculator will also save previous inputs to it's history and is able to show them in the console.

---

## How to

1. Open your terminal
Ensure that you have Node.js installed by typing node -v into the terminal

2. To add two numbers run the following (replacing a and b with your desired numbers)
console.add(a, b);
this will add a and b and save to history

3. To subtract two numbers run the following (replacing a and b with your desired numbers)
console.subtract(a, b);
this will subtract a and b and save to history

4. To Multiply two numbers run the following (replacing a and b with your desired numbers)
console.multiply(a, b);
this will multiply a and b and save to history

5. To Divide two numbers run the following (replacing a and b with your desired numbers)
console.divide(a, b);
this will multiply a and b and save to history

Note
If you attempt to divide by 0 a message will be displayed noting that you tried because this is an impossible operation
